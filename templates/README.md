# Templates

This folder will contain reusable prompt templates for AI-assisted engineering tasks.

Templates should be short, copy-pasteable, and easy to adapt inside a consumer project. They should define goal, scope, allowed files, forbidden files, verification, and expected output.

## Planned Templates

- `feature-plan.md`
- `backend-safe-slice.md`
- `frontend-safe-slice.md`
- `interrupted-session.md`
- `review-before-commit.md`
- `docs-release.md`
- `vpc-deploy-check.md`
