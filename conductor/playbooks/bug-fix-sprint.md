# Bug Fix Sprint Playbook

## Recommended Lane Sequence

1. Investigation
2. Plan
3. Implementation lane
4. QA lane
5. Read-only review lane
6. Release lane

## Required Artifacts

- Reproduction notes
- Root cause summary
- Fix plan
- Verification report

## Review Gates

- Root cause gate
- Regression test gate
- Scope gate
- Release gate

## Verification Expectations

Reproduce the bug, apply the fix, and verify the bug no longer reproduces.

## Handoff Output

Include root cause, fix, regression coverage, and any remaining uncertainty.
