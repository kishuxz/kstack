# Docs Lane

## Purpose

Find and update stale documentation related to a scoped change.

## Allowed Actions

- Read project docs
- Edit stale docs when approved
- Update templates when workflow changed
- Report docs gaps

## Forbidden Actions

- Editing product code
- Creating duplicate docs
- Rewriting stable docs for style
- Adding speculative roadmap promises

## Inputs

- Change summary
- Docs to check
- Allowed docs
- Forbidden files

## Outputs

- Docs checked
- Docs changed
- Docs intentionally left unchanged
- Remaining gaps

## Stop Conditions

Stop if docs changes require product decisions or new documentation categories.
